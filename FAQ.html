Version 1.0.0-beta

📝 Roadmap & Bug Tracker

Roadmap

1.) Combat Simulator Upgrade
    - While I'm going to keep the simple simulator, I intend to make a much more advanced and D&D-like combat simulator with spellcasting, save DCs, damage types, etc.
    - This will also be the backbone for a D&D combat tutorial for new players, and may become a full game!
    - This will be modular with JSON, so expect slow progress. The current priority is: Spellcasting and Saving Throws, Damage Types and Resistances, Conditions and Condition Immunities, Class Features, Subclass Features, Legendary Actions, AI Options, Importing Monsters from Core D&D Content, Feats, Miscellaneous Features
2.) Dice Rolling Updates
    - Improve 2D dice visuals (VFX)
    - There is currently no plan to include 3D dice and rolling physics
3.) Initiative Tracker Upgrade
    - Dungeon Masters can host a combat using the Initiative Tracker!
    - Players can join using a combat code, allowing them to see the initiative themselves and keep track of the order and their own stats
    - DM can hide monsters and monster stats using "fog of war"
4.) NPC Name generator
    - Include new part that generates names for dungeon masters
    - Can do quick name, or can have inputs to give name more personality
5.) Wild Magic Surge
    - Include Wild Magic Surge table and rolling
    - Allow feature for others to add to wild magic table (needs to be approved by me prior to working)
6.) Deck of Many Things
    - Include Deck of Many Things drawer
    - Use both decks, with a toggle to switch
    - Calculate probabilities of good vs. bad outcomes
7.) Miscellaneous
    - Speed up simulation times
    - Include tool-tips to help people use the site
    - Include a buy me a Ko-Fi
    - Include links to my other projects
    - Include ability for people to comment/suggest?
    - Move from local to web
    - Include special animations for Nat 1 and Nat 20
    - Include different dice in Skill Check Difficulty (1d100 for Divine Intervention)
    - Include extra conditions in Initiative Tracker (Exhaustion, Bless, etc.)
